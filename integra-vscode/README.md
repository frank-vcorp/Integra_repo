# INTEGRA Metodología — VS Code Extension

Instala y sincroniza automáticamente el sistema multi-agente INTEGRA en cualquier proyecto.

## Qué hace

- **Auto-sync**: Al abrir un proyecto INTEGRA, sincroniza skills e instructions automáticamente
- **Init Project**: Comando para inicializar un proyecto nuevo con estructura INTEGRA completa
- **Extensiones**: Instala GitHub Copilot y Qodo automáticamente
- **GitHub Apps**: Sugiere instalar CodeRabbit y Qodo Merge para revisión de PRs

## Comandos

- `INTEGRA: Inicializar Proyecto` — Crea estructura completa (PROYECTO.md, skills, instructions, etc.)
- `INTEGRA: Sincronizar Skills e Instructions` — Actualiza skills si hay cambios
- `INTEGRA: Verificar GitHub Apps` — Muestra instrucciones para instalar CodeRabbit y Qodo Merge
