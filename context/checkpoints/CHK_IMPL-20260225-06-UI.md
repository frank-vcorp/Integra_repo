### 2. Gestión de Citas (`/appointments`)
**Estado:** ✓ Completada

**Nueva Ruta:** `/appointments` (creada desde cero)

**Funcionalidad:**
- Lista todas las citas con `getAppointments()`
- Tabla con 6 columnas:
  - Trabajador (firstName, lastName, universalId)
  - Empresa (company.name)
  - Sucursal (branch.name)

- Botón "Check-in" → llama `checkInAppointment(appointmentId)`

- Solo mostrar botón Check-in si status === 'SCHEDULED'
- Error handling con UI toast rojo

   - Llama logAudit() automáticamente
