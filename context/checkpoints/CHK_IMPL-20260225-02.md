## 🔗 Endpoints Implementados

### Firma Digital

#### POST `/api/v1/sign-pdf`
Firma un PDF con certificado X.509 autofirmado.

```json
{
  "input_pdf": "documento.pdf",
  "output_pdf": "documento_signed.pdf",
  "reason": "Certificado Médico AMI",
  "password": "test1234"
}
```

**Respuesta:**
```json
{
  "status": "success",
  "message": "PDF firmado correctamente",
  "output_pdf": "/app/uploads/documento_signed.pdf",
  "signed_at": "2026-02-25T14:30:00",
  "signer": "AMI Test Signer"
}
```

#### POST `/api/v1/verify-signature`
Verifica la firma digital de un PDF.

```
POST /api/v1/verify-signature?file_path=documento.pdf
```

**Respuesta:**
```json
{
  "status": "success",
  "is_signed": true,
  "signatures_found": 1
}
```

### Reportes Masivos

#### POST `/api/v1/generate-report`
Genera reportes consolidados en múltiples formatos.

```json
{
  "data_list": [
    {"id": 1, "nombre": "John Doe", "tipo": "Audiometría", "resultado": "Normal"},
    {"id": 2, "nombre": "Jane Smith", "tipo": "Laboratorio", "resultado": "Anormal"}
  ],
  "formats": ["excel", "json", "html"],
  "title": "Reporte de Consolidación"
}
```

**Respuesta:**
```json
{
  "status": "success",
  "batch_id": "batch_20260225_143000",
  "generated_files": {
    "excel": "/app/reports/batch_20260225_143000.xlsx",
    "json": "/app/reports/batch_20260225_143000.json",
    "html": "/app/reports/batch_20260225_143000.html"
  },
  "timestamp": "2026-02-25T14:30:00"
}
```

#### POST `/api/v1/generate-excel-report`
Genera un reporte en Excel (versión especializada).

#### POST `/api/v1/generate-json-report`
Genera un reporte en JSON (versión especializada).

## 🏗️ Arquitectura de Servicios

### SignerService
```
Responsabilidades:
├─ Generar certificados autofirmados X.509
├─ Aplicar firmas digitales a PDFs
├─ Verificar validez de firmas
└─ Gestionar contraseñas de certificados

Características:
├─ Autocreación de cert si no existe (/app/certs/)
├─ Fallback a firma básica si pyHanko no está disponible
└─ Soporte para formatos PKCS#12 y PEM
```

### ReportService
```
Responsabilidades:
├─ Convertir listas de datos a reportes
├─ Generar Excel con formato básico
├─ Generar JSON estructurado
├─ Generar HTML para visualización
└─ Procesamiento en batch

Características:
├─ Auto-formato de columnas en Excel
├─ Timestamps automáticos
├─ Manejo de data vacía con errores claros
└─ Soporte múltiples formatos simultáneamente
```

## 🔐 Seguridad y Consideraciones

### Certificados de Prueba
- ⚠️ Certificados autofirmados SOLO para desarrollo/testing
- En producción, usar certificados de CAs reconocidas (RFC, GlobalSign, etc.)
- Contraseña por defecto "test1234" debe ser ambiental en producción

### Gestión de Secretos
- ✓ Contraseñas nunca hardcodeadas en logs
- ✓ API Keys almacenadas en variables de entorno
- ✓ Rutas de archivos sanitizadas with `os.path.basename()`

### Permisos de Archivos
- Reportes guardados en `/app/reports/` con permisos 0644
- Certificados en `/app/certs/` accesibles solo a la aplicación
