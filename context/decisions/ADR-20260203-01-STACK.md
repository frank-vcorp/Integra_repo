# ADR-20260203-01: Selección de Stack Tecnológico

## Contexto
El sistema "Residente Digital" requiere manejar flujos complejos de datos, digitalización de expedientes médicos, lectura de documentos (OCR) y una interfaz de usuario moderna y responsiva.

## Decisión
Se ha seleccionado la siguiente arquitectura híbrida:

1.  **Frontend (BFF)**: Next.js 14+ (App Router).
    -   *Justificación*: Permite interfaces ricas, gestión de estado robusta y optimización de rendimiento.
2.  **Backend (Core/IA)**: Python (FastAPI).
    -   *Justificación*: Ecosistema nativo para Inteligencia Artificial y OCR (Tesseract, PyTorch, etc.).
3.  **Base de Datos**: PostgreSQL.
    -   *Justificación*: Robustez para datos relacionales críticos (Expedientes Médicos).

## Estatus
Aprobado por el usuario.
